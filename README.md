## Моя идея для проекта:
< Тема: Создать ТГ бота на тему "Влияние глобального потепления на балых медведей". Бот будет рассказывать о причинах глобального потепления, рассказывать о проблеме и её влияния на белых медведей и возможных путях решения этой пробемы, а также немного интересных фактов касательно этого. В моём боте будет квиз проверяющий пользователя на знания этой темы. 

## Мой бот - "White Bear cub"
Управляется бот следующими командами:

   ✔️ /reason - причины возникновения глобального потепления в общем
  
   ✔️ /problem - рассказывает о проблеме 
  
   ✔️ /decisions - возможные пути решения этой проблемы
  
   ✔️ /fact - интересные факты на эту тему 
  
   ✔️ /quiz - тестирование на знание темы  

## Польза проекта:
< Да, бот не сможет решить проблему глобального потепления полностью, но за то сможет ознакомить вас с этой проблемой и направить на конкретные шаги по её минимизации.
