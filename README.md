## Моя идея для проекта:
< Тема: Создать ТГ бота на тему "Влияние глобального потепления на балых медведей". Бот будет рассказывать о причинах глобального потепления, рассказывать о проблеме и её влияния на белых медведей и возможных путях решения этой пробемы, а также немного интересных фактов касательно этого. В моём боте будет квиз проверяющий пользователя на знания этой темы.
